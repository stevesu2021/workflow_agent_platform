一、核心 AI 服务管理（已有 + 增强）
功能
说明
✅ 文本大模型管理
支持 Qwen、DeepSeek、Llama、ChatGLM、Moonshot、OpenAI 等
• 模型名称 / 别名
• API 地址（兼容 OpenAI 格式或自定义）
• API Key（加密存储）
• 最大上下文长度、temperature 默认值
• 是否启用流式响应
✅ 视觉大模型管理
如 Qwen-VL、GPT-4V、InternVL、CogVLM
• 支持图像输入格式（base64 / URL）
• 最大图像分辨率限制
• 是否启用多轮视觉对话
✅ OCR 服务管理
分为子类型：
• PaddleOCR（本地/远程部署）
• DeepSeek-OCR
• 阿里云 OCR / 百度 OCR（预留）
• 支持批量识别开关、语言类型（中/英/多语）
➕ 向量模型管理
用于 RAG 的 embedding 模型
• BGE、text2vec、OpenAI embeddings
• 维度（768/1024）
• 归一化开关
➕ 重排序模型（Reranker）管理
如 BGE-Reranker、Cohere Rerank
• Top-K 重排数量
• 相似度阈值
